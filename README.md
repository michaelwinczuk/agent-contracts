# Agent Contracts

Smart contract infrastructure for autonomous AI agent economies on EVM chains.

## Contracts

TollGate - Pay-per-call metering for agent services
AgentEscrow - Task-based escrow with deadline refunds  
SwarmSplitter - Revenue splitting across agent swarms

## Setup

npm install
npx hardhat compile
npx hardhat test

## 12 passing tests

## Stack

Solidity 0.8.20 / Hardhat / OpenZeppelin / Ethers.js
